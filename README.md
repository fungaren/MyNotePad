# MyNotePad

## 简介

一个支持 MarkDown 转换 HTML 的文本编辑器，仅一个 exe。

本项目的重点在于自己实现一个文本框。

[微软的例子](https://msdn.microsoft.com/en-us/library/windows/desktop/ms646268(v=vs.85).aspx)

## 已实现的功能 implemented

- 按 F5 快速在浏览器中渲染
- 支持黑白两色主题
- 支持行号显示
- 支持 Latex 渲染

## 未实现的功能 not implemented

- 撤销
- TAB 对齐
- 搜索、替换
- 编码转换（目前仅支持 UTF-8）
- 即时代码着色
- 彩色 Emoji
